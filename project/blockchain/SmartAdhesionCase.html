# 프로젝트 요약
중요한 시료를 쉽고 안전하게 보관하는 방법이 필요하다!!!
Smart Adhesion Case는 중요한 시료를 손쉽고 안전하게 보관 및 관리하기 위한 스마트 보관함이다.

# Smart Adhesion Case 목표
- 온도, 습도, 가스 측정이 24시간 진행되야 한다.
- 이상이 생길 경우 임시 대처 및 경고 메시지와 함께 로그를 남겨둔다.
- 얼굴 인식을 통하여 인가된 사용자만 잠금장치가 해제되야 한다.
- 관리 페이지를 통하여 상태 확인 및 잠금장치 해제가 가능해야 한다.
- 보관함 자체의 이상을 사용자가 확인하기 쉬워야 한다.

# 기술 스택
Next.js / HyperledgerFabric / OpenCV / Raspberry Pi / Grafana / Python / CouchDB / Docker / HTML
