# 프로젝트 요약
안전하게 사용할 수 있는 투표 시스템을 구축할 수 없을까?
Hyperledger fabric을 활용하여 저장된 데이터의 조작이 불가능에 가까운 투표 시스템이다.

# Blockchain Vote 목표
- 비밀선거의 원칙을 지켜야 한다.
- 투표의 현황을 실시간으로 확인할 수 있어야 한다.
- 자신의 투표를 알아낼 수 있어야 한다.
- 투표방을 만들 수 있어야 한다.
- 후보의 정보를 쉽게 확인할 수 있어야 한다.

# 기술 스택
Next.js / HyperledgerFabric / Grafana / CouchDB / Docker / HTML
